# Installation

## Stable release

To install SCIN Data Modeling, run this command in your terminal:

```sh
uv add scin-data-modeling
```

Or if you prefer to use `pip`:

```sh
pip install scin-data-modeling
```

## From source

The source files for SCIN Data Modeling can be downloaded from the [Github repo](https://github.com/wildg/scin_data_modeling).

You can either clone the public repository:

```sh
git clone https://github.com/wildg/scin_data_modeling
```

Or download the [tarball](https://github.com/wildg/scin_data_modeling/tarball/main):

```sh
curl -OJL https://github.com/wildg/scin_data_modeling/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd scin_data_modeling
uv sync
```
